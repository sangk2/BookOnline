Trước khi run ứng dụng
I. Thiết lập server
	1. Mở thư muc server trong BookOnline
	2. Copy thư mục android-network vào htdocs của xampp
	3. Khởi động xampp chạy Apache và MySQL
	4. Truy cập link "http://localhost/phpmyadmin/" bằng trình duyệt
	5. Import dữ liệu từ file "book_online.sql" trong thư mục vào phpMyAdmin
II. Đổi IP wifi
	1. Vào cài đặt của wifi và xem "IPv4 address" của wifi
	2. Mở file IPwifi.java trong app tiến hành thay đổi địa chỉ IP
III. Chạy ứng dụng
	Sau khi thiết lập server và đổi IP wifi thì Run để xem kết quả
